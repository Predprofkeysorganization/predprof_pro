Инструкция по установки:
1) Перенесем к себе на устройства: "git clone -b master --single-branch https://github.com/Predprofkeysorganization/predprof_pro.git"
2) открываем проект в среде разработки.
3) Установим фреймворк и другие библиотеки "pip install -r requerements.txt"
4) Открывам терминал в среде разработки и используем команду "python3 manage.py runserver"
